# java-pro-homework

## Описание
Проект для выполнения домашних заданий по чистой Java курса Java PRO от Академии Т1.

## Стек используемых технологий
* Java 17
* Apache Maven 3.8.2
* JUnit 5.10.2
* Checkstyle 10.7.0

## Требуемое окружение для запуска
* JDK 17
* Apache Maven 3.8
